🚀 Quer salvamento automático? Clique aqui!

Para ativar o salvamento automático, use um servidor web local:

Método 1 - Live Server (VS Code):

  1. Abra a pasta do projeto no VS Code
  2. Instale a extensão "Live Server"
  3. Clique direito no index.html → "Open with Live Server"
  4. Acesse http://localhost:5500

Método 2 - Python (já instalado no Windows):

  1. Abra o Prompt de Comando na pasta do projeto
  2. Execute: python -m http.server 8000
  3. Acesse http://localhost:8000

Método 3 - Node.js:

  1. Instale: npm install -g http-server
  2. Execute: http-server na pasta do projeto
  3. Acesse http://localhost:8080

✨ Com servidor local, o botão "Configurar Pasta" aparecerá e você terá salvamento automático!
